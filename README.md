# Nexus: Financial Liquidity Crisis Early Warning System

**Accepted at IEEE MIT URTC 2025 — Main Technical Track, Technology for Humanity**

A probabilistic machine learning early warning system that detects financial liquidity crises in U.S. banks before they escalate, integrating regulatory filings, regional macroeconomic indicators, and labor-market signals to deliver calibrated, explainable risk estimates each quarter.

For the full project writeup — architecture, ML contributions, results tables, and reproduction instructions — see [`nexus/README.md`](./nexus/README.md).

## What's in this repo

- [ ] `nexus/` — the complete project: source code, notebooks, configs, and the in-depth README with full methodology and results
